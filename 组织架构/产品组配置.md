# 产品组配置 #

### 一、产品组定义

1. SPRO进入IMG
2. 企业结构=》定义=》物流，常规=》→定义、复制、删除、检查产品组
3. 进入产品组列表
4. 定义新的产品组
![更改视图_产品组_概览](/images/组织架构/更改视图_产品组.jpg "更改视图_产品组")

* 产品组使用两位编码，可用数字和字母

 

### 二、产品组-销售组织分配
1. SPRO进入IMG
2. 企业结构=》分配=》销售与分销=》给销售组织分配部门
3. 选择销售组织和产品组
![更改视图_分配销售组织-产品组](/images/组织架构/更改视图_分配销售组织-产品组.png "更改视图_分配销售组织-产品组")

* 一个产品组可以分配给多个销售组织
* 多个产品组可以分配给一个销售组织
* 销售组织和产品组为多对多关系
