# 物料主数据 #

### 物料数据分类
* 基础数据
    * 与所有区域相关 
* 销售：销售组织
    * 与销售和分销相关，对各自的销售和分销的渠道有效
* 销售：一般/工厂
    * 与交货装运相关
* 采购数据
* 工程/设计详细数据
* 会计
* 成本核算
* 仓库管理
